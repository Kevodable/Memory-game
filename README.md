# Memory Game

Gioco delle coppie (memory) realizzato in HTML, CSS e JavaScript puro.

## Come giocare

Apri `index.html` in un browser. Scegli la difficoltà (4x4 o 6x6) e trova tutte le coppie di emoji nel minor numero di mosse e nel minor tempo possibile.

## Funzionalità

- Conteggio mosse e timer
- Due livelli di difficoltà
- Animazioni di flip e feedback per le coppie sbagliate
- Schermata di vittoria con riepilogo finale
